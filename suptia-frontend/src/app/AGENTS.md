# AGENTS — ページ実装

## ページごとの要件
- `/`: 検索バー + 診断CTA、いま安い/人気/注意成分セクション
- `/diagnosis`: 3〜5問フォーム、進捗バー、送信で`/results`へ
- `/results`: 推奨カード（実効コスト/日・アラート・4スコア）、並び替え機能
- `/products/[slug]`: 成分配合、正規化価格比較、相互作用・禁忌、研究要約、価格履歴

## 非機能要件
- next/image使用（width/height必須）
- エラー/ローディングUI必須
- CTAボタンにaria-label付与
- 各ページに構造化データを埋め込み

## 禁止事項
- 薬機法NG表現をUIテキストに含める
- アラートやスコアの非表示
